## Requirements

## Getting Started

1. Install dependencies by running:<br><br>
When using npm:<br>
```npm install```<br><br>
When using yarn:<br>
```yarn install```<br>

2. Run the GraphQL server: ```npm run start```<br>
If everything is okay you will see the url on which you can access the GraphQL playground
